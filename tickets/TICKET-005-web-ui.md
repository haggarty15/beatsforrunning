# TICKET-005: Web Interface (Flask UI)

## Description

Create a Flask-based web interface that allows users to input their running pace and music preferences, and generates a playlist using the core logic. Ensure the UI is modern and responsive.

## Acceptance Criteria

- [x] Flask application set up in `src/beatsforrunning/`
- [x] Routes for home page and playlist generation
- [x] Integration with `core/tempo.py`, `connectors/spotify.py`, and `core/playlist.py`
- [x] Modern HTML/CSS frontend (reusing/improving existing `index.html`)
- [x] BDD tests passing for UI interactions
- [x] Follows project standards (src layout, dependency management)
